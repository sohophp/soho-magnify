SOHO-MAGNIFY 图片放大镜

1.使用放大鏡


                  $('img.example-img').SohoMagnify({
                      glass:1
                  });

2.不使用放大鏡


                  $('img.example-img').SohoMagnify({
                      glass:0
                  });

3.不使用放大鏡加光標

                  $('img.example-img').SohoMagnify({
                      glass:1,
                      cursor:'move'
                  });
ECMAScript 6 的 module 版本 支持IE通用版本
NPM
npm i soho-magnify
GIT
git clone git@github.com:sohophp/soho-magnify
github.com项目网址
https://github.com/sohophp/soho-magnify

使用方法

<link type="text/css" rel="stylesheet" href="src/css/soho-magnify.css"/>
<script src="external/jquery.min.js"></script>
<script src="src/js/soho-magnify.js"></script>
<script>
    $(window).on('load', function () {
        $('img.example-img').SohoMagnify();
    });
</script>
